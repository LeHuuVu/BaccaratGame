# BaccaratGame
 Baccarat with Python

# First step:
 Run Entity.py file to create a new database

# Required Module:
 ## Fastapi
    pip install fastapi
 ## Uvicorn (Asynchronous Server Gateway Interface)
    pip install uvicorn
---
 Run server:
    python -m uvicorn api:app --reload --host localhost
 ## Pydantic
    pip install pydantic
